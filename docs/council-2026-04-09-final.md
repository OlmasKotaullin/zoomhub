## Консенсус (все 4 эксперта)

1. **Рынок доказан Буквицей.** 35K платящих, 120M₽/год чистыми. Не гипотеза.
2. **MVP = Telegram-бот, не веб.** Бот снижает порог входа до нуля, даёт виральность.
3. **Безопасность — до любого запуска.** IDOR, токены, invite bypass — 1.5 дня.
4. **Не тратить время Алмаза на ручной growth.** SEO на автопилоте + сарафан.
5. **Расползание скоупа — убийца.** Скиллы, память, поиск — всё v2.

## Telegram-бот: архитектура

- Часть main FastAPI app (не отдельный процесс)
- Webhook уже есть, расширяем
- Flow: аудио в бот → RunPod транскрибирует → LLM саммари → результат в Telegram + в БД
- Линковка: /start <jwt> (уже реализовано)
- Лимит 20MB: достаточно для большинства файлов, для больших → "загрузите через сайт"
- Трудозатраты: 2 дня
