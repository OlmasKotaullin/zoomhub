Установка на Mac

macOS может заблокировать приложение. Вот как это исправить:

1

Скачайте и распакуйте

ZIP скачается автоматически. Safari распакует его — в Загрузках появится ZoomHubAgent.app

2

Двойной клик на ZoomHubAgent.app

Появится предупреждение macOS — нажмите «Готово»

«ZoomHubAgent не может быть открыт, т.к. разработчик не проверен»

Это нормально — Apple не знает наше приложение. Исправим на следующем шаге.

3

Откройте Системные настройки

 (яблоко в углу) → Системные настройки → Конфиденциальность и безопасность

Прокрутите вниз — увидите сообщение: «Использование ZoomHubAgent было заблокировано»

4

Нажмите «Подтвердить вход»

Введите пароль Mac, если попросит. Затем снова запустите ZoomHubAgent.app.

Готово! Приложение откроет браузер для настройки. Повторная блокировка не произойдёт.

У вас Windows — этот шаг не нужен. Нажмите «Далее».

AI для конспектов

Необязательно

Выберите нейросеть для создания конспектов. По умолчанию — Groq (бесплатно).

Groq · Llama 3.3 70B Рекомендуем

Бесплатно, сверхбыстрый, контекст ~2 часа. Английская модель, но хорошо понимает русский.

Gemini 2.5 Flash Бесплатно

Google AI. Огромный контекст ~15 часов. Отлично для длинных встреч.

GigaChat (Сбер) 1М токенов/мес

Русскоязычный AI от Сбера. Контекст ~1.5 часа. Лучше всех понимает русский.

Claude Sonnet Платный

Anthropic. Лучшее качество конспектов, но требует оплату (~$3/М токенов).

Получите API-ключ Groq:

  1. Откройте console.groq.com/keys
  2. Войдите через Google или GitHub
  3. Нажмите Create API Key
  4. Скопируйте ключ (начинается с gsk_)
